# 🤖 Automação RPA 

Este projeto é um **RPA (Robotic Process Automation)** desenvolvido em **Java**, com o objetivo de automatizar tarefas repetitivas e reduzir erros humanos.

- **Linguagem:** Java 8   
- **Frameworks:** Selenium.  
- **Build:** Maven
